# kht-hypertap-test
Hypertap test written in python and pygame-ce. It tests how fast you can hyper tap in 5 secs.
* To use it, type git clone https://github.com/Kavi-Source-Code/kht-hypertap-test.git in terminal.

* Background:


![Background](Background.png)

## Before using, make sure to type in terminal: pip install pygame-ce OR pip3 install pygame-ce OR python -m pip install --user pygame-ce OR python -m pip3 install --user pygame-ce

* The text on the left is how much time you've taken and the text on the right is how many times you tapped.

* You can use this score chart:

    10-14: Poor
    
    15-19: Needs practice
    
    20-24: Acceptable
    
    25-29: Good
    
    30-34: Great
    
    35-40: Excellent
